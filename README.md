# Crystal Viewer

## Run Project:
```bash
# Quick start:
python -m http.server 8000
# then open: http://localhost:8000

# Or:
npx serve .
```

## Quick Run Options:
1. **With local server (recommended):**
   ```bash
   # Python
   python -m http.server
   
   # Node.js
   npx serve .
   ```

2. **Direct (limited features):**
   - Just open `index.html` in browser
   - Note: Upload files only (no default models)

## Access:
- Open browser to `http://localhost:8000`
- Upload GLB files or use demo cube
- Use GUI controls to adjust materials
